# Facilitation Playbook

## Opening technique

Open with a simple framing question such as:

> If a network stays online but leaders trust the wrong information, has the operation still succeeded?

This sets up the class to think beyond uptime and outage.

## Recommended facilitation methods

### Guided discussion

Ask learners to interpret incidents before you give the answer.

### Whiteboard mapping

Map each case to:

- physical effect
- informational effect
- cognitive effect
- strategic consequence

### Live browsing

Use public sources to show learners that cyber concepts are observable in real reporting, threat analysis, doctrine, and incident response material.

### Pause-and-translate method

When a slide contains dense wording, stop and translate it into ordinary English before continuing.

## Good prompting questions

- What was the attacker really trying to achieve?
- What did the target lose: data, trust, time, control, or credibility?
- Which dimension mattered most in this case?
- Was the technical action the objective, or only the mechanism?
