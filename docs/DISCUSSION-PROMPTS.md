# Discussion Prompts

## Prompt set 1 — Dimensions of cyberspace

- Can an operation begin in one dimension and matter most in another?
- Which dimension is most visible to the public during a cyber incident?
- Which dimension is easiest to underestimate?

## Prompt set 2 — Information superiority

- Is information superiority about having more data, or better understanding?
- Can speed outweigh accuracy in contested environments?
- What happens when both sides have incomplete information?

## Prompt set 3 — Influence and cognition

- Why do false narratives often spread faster than corrections?
- How does trust act as a battlespace variable?
- Can perception damage be more strategically significant than technical damage?

## Prompt set 4 — Incident analysis

- Which case best shows cyber-physical effect?
- Which case best shows decision disruption?
- Which case best shows information dominance as a strategic advantage?
