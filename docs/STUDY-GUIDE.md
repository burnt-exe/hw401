# Study Guide

## What to revise first

Start with these anchor concepts:

- fifth domain
- physical, informational, cognitive dimensions
- information superiority
- influence operations
- decision-loop disruption
- cyber-physical effects

## Best way to study

Do not memorize terms in isolation. Instead:

1. learn the concept in simple language  
2. link it to a real incident  
3. explain what the effect was  
4. identify which dimension was most affected

## Strong revision question set

- What was the attacker trying to gain?
- What changed for the target?
- Was the effect technical, informational, cognitive, or all three?
- Was the cyber action the end goal or the delivery mechanism?

## Exam mindset

Strong answers usually connect a concept to a consequence. Weak answers only define terms without showing why they matter.
