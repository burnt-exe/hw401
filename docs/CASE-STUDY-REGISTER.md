# Case Study Register

## Priority incidents to add

### Stuxnet

Use for cyber-physical effect, ICS targeting, covert sabotage, and strategic delay.

### Ukraine power grid incidents

Use for operational disruption, infrastructure targeting, and public impact.

### NotPetya

Use for destructive effects, spillover, economic damage, and strategic ambiguity.

### SolarWinds

Use for stealthy access, espionage scale, trust in software supply chains, and systemic exposure.

### Colonial Pipeline

Use for indirect societal impact, critical infrastructure dependency, and decision under pressure.

## Suggested analysis template

For every case study document, capture:

- incident summary
- target and environment
- attack mechanism
- affected dimensions
- operational outcome
- strategic significance
- teaching notes
